문제 7) : 세준이는 자기 점수 중에 최댓값을 골랐다. 이 값을 M이라고 한다. 그리고 나서 모든 점수를 점수/M*100으로 고쳤다. 세준이의 성적을 위의 방법대로 새로 계산했을 때, 새로운 평균을 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 시험 본 과목의 개수 N이 주어진다. 이 값은 1,000보다 작거나 같다. 둘째 줄에 세준이의 현재 성적이 주어진다. 이 값은 100보다 작거나 같은 음이 아닌 정수이고, 적어도 하나의 값은 0보다 크다.

문제 8) : OOXXOXXOOO와 같은 OX퀴즈의 결과가 있다. O는 문제를 맞은 것이고, X는 문제를 틀린 것이다. 문제를 맞은 경우 그 문제의 점수는 그 문제까지 연속된 O의 개수가 된다. OX퀴즈의 결과가 주어졌을 때, 점수를 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 테스트 케이스의 개수가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 길이가 0보다 크고 80보다 작은 문자열이 주이진다. 문자열은 O와 X만으로 이루어져 있다.

출력 : 각 테스트 케이스마다 점수를 출력한다.

제한 : X

정답 결과 추가
N = int(input())

for i in range(N):
    OX = list(input())
    score = 0
    sum_score = 0 # 새로운 리스트를 입력 받으면 점수 합계를 리셋한다.
    for k in OX:
        if k == 'O':
            score += 1 # 'O'가 연속되면 점수가 1점씩 커진다.
            sum_score += score
        else:
            score = 0
    print(sum_score) 		입력
1
OOXXOXXOOO
출력
10 		- 우선 반복문을 통해 OX퀴즈를 입력받아야 한다. 이때, 'O'일 경우 1을 더해주고 모든 점수를 합한 변수 2개를 0으로 설정한다. 이는 새로운 리스트가 들어왔을 때 0으로 리셋된다.
- 그 다음 각 리스트마다 'O'의 여부를 묻고 점수를 올려주고 각 더해준다.

문제 9) : 대학생 새내기들의 90%는 자신이 반에서 평균은 넘는다고 생각한다.

입력 : 첫째 줄에는 테스트 케이스의 개수 C가 주어진다. 둘째 줄부터 각 테스트 케이스마다 학생의 수 N(1 <= N <= 1000, N은 정수)이 첫 수로 주어지고, 이어서 N명의 점수가 주어진다. 점수는 0보다 크거나 같고, 100보다 작거나 같은 정수이다.

출력 : 각 케이스마다 한 줄씩 평균을 넘는 학생들의 비율을 반올림하여 소수점 셋째 자리까지 출력한다.

제한 : X

정답 결과 추가
C = int(input())

for i in range(C):
    lt1 = list(map(int, input().split()))
    avg = sum(lt1[1:])/lt1[0] # 평균을 구함(lt1[0] : 학생수, lt1[1:] : 점수)
    cnt = 0
    for k in lt1[1:]:
        if k > avg:
            cnt += 1
    rate = cnt/lt1[0] * 100
    print(f"{rate:.3f}%") 		입력
1
5 50 50 70 80 100
출력
40.000% 		- 인덱스 활용
