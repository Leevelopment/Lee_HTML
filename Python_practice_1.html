문제 6) : 두 자연수 A와 B가 주어진다. 이때, A + B, A - B, A * B, A / B(몫), A % B(나머지)를 출력하는 프로그램을 작성하시오.

입력 : 첫째 줄에 A와 B가 주어진다. (1 <= A,B <= 10,000)

출력 : 첫째 줄에 A + B, 둘째 줄에 A - B, 셋째 줄에 A * B, 넷째 줄에 A / B, 다섯째 줄에 A % B를 출력한다.

정답 결과 추가
A,B = input().split()
print(int(A) + int(B))
print(int(A) - int(B))
print(int(A) * int(B))
print(round(float(A) / float(B), 0))
print(float(A) % float(B)) 		입력
7 3
출력
10 4 21 2.0 1.0 		- 나누기 소수점 제외 -> round() 사용
- 출력과 같이 표현하려면 float말고 int로 사용하면 됨(단, 나누기 부분은 전체부분의 int 사용)

출력 : 첫째 줄에 준하의 놀람을 출력한다. 놀람은 아이디 뒤에 ??!를 붙여서 나타낸다.

문제 9) : 체스는 총 16개의 피스를 사용하며, 킹 1개, 퀸 1개, 룩 2개, 비숍 2개, 나이트 2개, 폰 8개로 구성되어 있다. 흰색 피스의 개수가 주어졌을 때, 몇 개를 더하거나 빼야 올바른 세트가 되는지 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 찾은 흰색 킹, 퀸, 룩, 비숍, 나이트, 폰의 개수가 주어진다. 이 값은 0보다 크거나 같고 10보다 작거나 같은 정수이다.

출력 : 첫째 줄에 입력에서 주어진 순서대로 몇 개의 피스를 더하거나 빼야 되는지를 출력한다. 만약 수가 양수라면 그 개수 만큼 피스를 더해야 하는 것이고, 음수라면 제거해야 하는 것이다.

정답 결과 추가
chess = [1, 1, 2, 2, 2, 8] # 받을 때 리스트로 받는다
x_chess = list(map(int, input().split())) # 많이 쓰는 방법 -> 입력받은 여러 숫자를 정수형으로 리스트에 담는다

for i in range(6):
    print(chess[i] - x_chess[i], end=" ") # 전체 말의 개수에서 입력받은 말의 개수 차감 		입력
0 1 2 2 2 7
출력
1 0 0 0 0 1 		- 우선 전체 체스 말 수를 입력할 변수가 필요하다.
- 그 다음 입력받은 말의 개수에 따라 차감 또는 증가해주어야 한다.
- 각 결과를 띄어쓰기로 표현

문제 10) : (A+B) % C는 ((A%C) + (B%C)) % C와 같을까?
(AXB) % C는 ((A%C) X (B%C)) % C와 같을까?
세 수 A,B,C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

입력 : 첫쨰 줄에 A,B,C가 순서대로 주어진다. (2 <= A, B, C <= 10000)

출력 : 첫째 줄에 (A+B) % C, 둘째 줄에, ((A%C) + (B%C)) % C, 셋쨰 줄에 (AXB) % C, 넷째 줄에 ((A%C) X (B%C)) % C를 출력한다.

정답 결과 추가
A,B,C = map(int,input().split())
print((A+B) % C)
print(((A%C) + (B%C)) % C)
print((A*B) % C)
print(((A%C) * (B%C)) % C) 		입력
5 8 4
출력
1 1 0 0 		- map(int, input().split()) 많이 사용

문제 11) : (그림참고) (1)과 (2)위치에 들어갈 세 자리 자연수가 주어질 때 (3), (4), (5), (6) 위치에 들어갈 값을 구하는 프로그램을 작성하시오.

입력 : 첫째 줄에 (1)의 위치에 들어갈 세 자리 자연수가, 둘째 줄에 (2)의 위치에 들어갈 세자리 자연수가 주어진다.

출력 : 첫째 줄부터 넷째 줄까지 차례대로 (3),(4),(5),(6)에 들어갈 값을 출력한다.

정답 결과 추가
num1 = int(input())
num2 = input()

print(num1 * int(num2[2]))
print(num1 * int(num2[1]))
print(num1 * int(num2[0]))
print(num1 * int(num2)) 		입력
472
385
출력
2360 3776 1416 181720 		- 각 자리 숫자의 곱으로 이루어져야 한다.
- 첫번쨰 숫자는 정수형, 두번쨰 숫자는 문자열 그대로 받은 후, 인덱스를 활용하여 백의 자리수, 십의 자리수, 일의 자리수를 각각 곱하여준다.
- 인덱스 값은 문자형이므로, 정수형으로 만든 후 곱해주어야 한다.
